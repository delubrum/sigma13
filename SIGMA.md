# SIGMA PROJECT ARCHITECTURE RULES

- **SERVER:** FrankenPHP (Worker Mode / Octane).
- **DB:** PostgreSQL 18 (Native JSONB for RBAC & Metadata).
- **STACK:** HALT (HTMX 2.x, Alpine.js, Laravel 13, Tailwind 4).
- **PHILOSOPHY:** KISS (Keep It Simple, Stupid). No libraries without approval.


## 1. CORE PHILOSOPHY: KISS & SOBRIETY
* **Principle:** Low technical complexity, high operational efficiency.
* **UI/UX:** Industrial Aesthetic (Grayscale, high data density, sober design, Dark mode toggle).
* **Language:** Code and technical docs in **English**. UI and system messages in **Spanish**.
**ALWAYS USE GLOBAL HTMX ORCHESTATORR**
